## 📌 프로젝트 개요

### 인원

| 이름 | 역할 |
| --- | --- |

| 유명현 | 팀원 |

### 🎯목표
- 서보 모터(조향 장치)를 통한 앞바퀴 방향 제어
    - 조향 장치를 사용하는 이유 : 조향장치는 회전 시 미끄러짐과 마모를 줄이고, 실차와 동일한 운동학으로 정확한 궤적 추종과 고속 안정성을 확보하기 위한 필수 요소다.
- 블루투스 통신을 통한 RC카 주행 시스템
- 초음파 센서를 통한 충돌 방지 시스템
- 스마트 주차장 시스템

### 담당 파트

| 최다루한 | APP 인벤터 개발, 스마트 주차장 구현, 자율 주행 프로토타입 함수 구현 |
| 유명현 | RC카 조향 기능 구현(코드, 하드웨어), 하드웨어 제작, RC카 코드 최적화, 자율 주행 시스템 구현 |

### RC카 회로도
![Image](https://github.com/user-attachments/assets/be05bd03-b0e2-4490-b675-361fc7a7a347)
